# 배터리 내부저항 계산기 - 직류부하법 (DC Load Method)

직류부하법(DC Load Method)을 사용한 정밀한 배터리 내부저항 측정을 위한 Windows용 프로그램입니다.

## 직류부하법이란?

직류부하법은 **IEC 61951-1**, **IEEE Std 1188** 등 국제 표준에서 규정된 배터리 내부저항 측정 방법입니다.

### 측정 원리
1. **개방전압(OCV)** 측정: 무부하 상태에서 배터리 단자전압 측정
2. **직류 부하 연결**: 일정한 저항값의 부하를 배터리에 연결  
3. **안정화 대기**: 과도응답이 끝날 때까지 충분히 대기
4. **부하 상태 측정**: 부하 단자전압과 부하전류 측정

### 계산 공식
```
R_internal = (V_OCV - V_load) / I_load
```

여기서:
- **V_OCV**: 개방전압 (Open Circuit Voltage)
- **V_load**: 부하 단자전압
- **I_load**: 부하전류
- **R_internal**: 내부저항

## 주요 기능

### 1. 직류부하법 정밀 계산
- 국제 표준 직류부하법 공식 적용
- 측정된 전류와 계산된 전류 비교 검증
- 키르히호프 전압 법칙을 이용한 계산 검증
- 4단계 계산 과정 상세 표시

### 2. 측정 정확도 향상
- **전류 직접 측정 지원**: 암페어미터로 측정한 값 입력 가능
- **과도응답 제거**: 배터리별 최적 안정화 시간 가이드
- **측정값 검증**: 계산값과 측정값 비교 분석
- **오차 분석**: 측정 오차 원인 및 해결 방법 제시
