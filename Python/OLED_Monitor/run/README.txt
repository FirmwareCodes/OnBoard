OnBoard OLED Monitor - 독립 실행 파일
==========================================

이 폴더에는 OnBoard OLED Monitor의 독립 실행 파일이 포함되어 있습니다.

파일 설명:
- OnBoard_OLED_Monitor.exe : 메인 프로그램 (독립 실행 파일)
- start_monitor.bat        : 프로그램 시작 스크립트
- README.txt               : 이 파일

사용 방법:
1. OnBoard_OLED_Monitor.exe를 직접 실행하거나

시스템 요구사항:
- Windows 10/11 (64-bit)
- 시리얼 포트 드라이버 설치됨

문제 해결:
- 프로그램이 실행되지 않으면 Windows Defender 예외 목록에 추가하세요
- 연결 문제시 다른 프로그램이 포트를 사용 중인지 확인하세요
